# AICO

AICO is a platform for building AI-powered voice agents. It combines a Bun-native backend, Svelte 5 frontend, LiveKit WebRTC integration, Python agent workers, and local STT/TTS services.

## Platform Components

<CardGroup cols={2}>
  <Card title="Flow Builder" icon="diagram-project" href="/flows/overview">
    Visual workflow editor for designing conversation flows with branching logic, tool calls, and memory management.
  </Card>
  <Card title="Voice Agents" icon="microphone" href="/agents/overview">
    Real-time voice processing with LiveKit, supporting inbound and outbound telephony via SIP.
  </Card>
  <Card title="Connectors" icon="plug" href="/connectors/introduction">
    REST API integrations for external systems like workshop management and booking platforms.
  </Card>
  <Card title="Provider System" icon="cloud" href="/providers/overview">
    Unified interface for LLM, STT, and TTS providers (OpenAI, Anthropic, Groq, Vosk, Piper, etc.).
  </Card>
</CardGroup>
